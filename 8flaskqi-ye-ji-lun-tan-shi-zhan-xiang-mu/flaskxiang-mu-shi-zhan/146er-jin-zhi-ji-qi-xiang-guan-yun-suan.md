### 2.二进制转十进制

```
# 将十进制的n转为二进制的m
import math
n = int(input())
print("十进制:",n)
strnum = ""
while n>0:
    strnum += str(math.floor(n%2))
    n = math.floor(n/2)
print("二进制:",strnum[::-1])
```

### 3.二进制之间的与\(&\)或\(\|\)运算

```
与运算:a&b,只有a，b为1时，结果才为1
或运算:a|b，只要a或b中有一个为1，结果就为1
```

### 要判断某个用户有没有权限\(a\)

只要将对比的用户的权限和a权限的二进制码进行与运算，如果得到的结果和a相等，就代表这个用户有a这个权限，否者代表没有。
