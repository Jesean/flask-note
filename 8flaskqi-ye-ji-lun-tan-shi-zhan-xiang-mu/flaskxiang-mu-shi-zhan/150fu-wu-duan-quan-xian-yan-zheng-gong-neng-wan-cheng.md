为了防止跨越权限访问，还需要做一层验证

```
# 权限验证
def permission_requeired(permission):
    def outter(func):
        @wraps(func)
        def inner(*args,**kwargs):
            user = g.cms_user
            # 判断用户是否具有访问权限
            if user.has_permission(permission):
                return func(*args,**kwargs)
            else:
                return redirect(url_for('cms.index'))
        return inner
    return outter
```

这里所进行的验证，是基于原来的规定的功能的二进制码，进行与或运算判断是否具有相应的访问权限
