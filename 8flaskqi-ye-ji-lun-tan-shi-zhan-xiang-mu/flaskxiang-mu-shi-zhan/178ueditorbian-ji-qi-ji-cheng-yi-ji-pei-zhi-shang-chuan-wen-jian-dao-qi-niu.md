### 5.后台

```
from flask import Flask,render_template
import qiniu,config
from ueditor import bp

app = Flask(__name__)
app.register_blueprint(bp)
app.config.from_object(config)

@app.route('/')
def hello_world():
    return render_template("index.html")


if __name__ == '__main__':
    app.run(debug=True)

```
