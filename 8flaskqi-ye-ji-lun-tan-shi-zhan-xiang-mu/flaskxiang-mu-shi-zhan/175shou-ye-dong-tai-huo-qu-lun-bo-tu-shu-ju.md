前台视图

```
@bp.route('/')
def index():
    banners = BannerModel.query.order_by(BannerModel.priority.desc(),BannerModel.create_time.desc()).limit(4)
    context = {
        "banners":banners,
    }
    return render_template("front/index.html",**context)
```
