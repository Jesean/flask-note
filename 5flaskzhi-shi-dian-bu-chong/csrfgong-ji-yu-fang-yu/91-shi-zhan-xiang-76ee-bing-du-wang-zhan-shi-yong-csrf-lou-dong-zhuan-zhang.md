## stealer.py

```text
from flask import Flask,render_template

app = Flask(__name__)


@app.route('/')
def index():
    return render_template('index.html')

@app.route('/transfer/')
def transfer():
    return render_template('transfer.html')


if __name__ == '__main__':
    app.run(debug=True,port=5001)
```
